## 📦 Installation Guide

1. **Backup Current Template**
   ```bash
   Dashboard → Theme → Backup
   ```

2. **Install New Template**
   ```bash
   Dashboard → Theme → Edit HTML → Copy index.xml content
   ```

3. **Configure Settings**
   ```bash
   Dashboard → Theme → Customize → Apply settings
   ```
